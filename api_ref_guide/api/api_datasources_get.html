List All Data Sources

Lists the data sources currently configured in Platfora. This call returns an array of data source objects.

URL

/api/v1/datasources

HTTP Method

GET

URL Parameters

None supported.

Data Parameters

None for GET requests.

Sample Call

For GET requests, the body of the HTTP request should be empty.

Success Response

Returns all data sources defined in Platfora. The example below shows one of each type.
