vizboards Resource
Parent topic: vizboards Resource

Overview

Use this resource to list, create, or export visualizations.

Reference title

A vizboard is where data analysts create and manage visualizations around a particular project or subject area. A vizboard can contain one or more pages. Pages can contain one or more visualizations. Each visualization can reference a different lens, and each lens can reference one or more datasets.

Attribute

Description

vizboardDetails

Contains a structure describing the vizboard pages and their contents (worksheets).

datasets

Contains an array of the datasets objects. The focus datasets reference these datasets.

lenses

Contains the array of lenses that provide data to visualizations on the board.

focusDatasets

Contains an array of the focus datasets. These focus datasets provide the basis for the lenses.

history

Contains a history object array. This array shows the boards change history.

href

Contains the board's URI.

webuiInternal

Contains a structure documenting the displayed titles for the visualization and its pages.

The vizboardDetails structure has the following:

Attribute

Description

pages

Contains array of page objects. Pages contain additional structures that describe it. Include is a panes array, one for each visualization on the page.

worksheets

Contains an array of visualization worksheets.

activePageIndex

Identifies the currently active page.

version

Contains a Platfora tracked value.

title

Contains the board's title.

localFields

 Contains an array of the visualization fields.
