List All Lenses

Use the GET method to retrieve and list the lenses currently configured in Platfora.

URL

/api/v1/lenses

HTTP Method

GET

URL Parameters

None supported.

Data Parameters

None for GET requests.

Sample Call

For GET requests, the body of the HTTP request should be empty.

Success Response

This response returns all lenses defined in Platfora.

HTTP status return code: 200
