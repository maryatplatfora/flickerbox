List Vizboards

Use the GET method to retrieve and list the vizboards currently configured in Platfora. Platfora recommends that the only vizboard field you edit is title inside vizboardDetails.

URL

/api/v1/vizboards

HTTP Method

GET

URL Parameters

None supported.

Data Parameters

None for GET requests.

Sample Call

For GET requests, the Body of the HTTP request should be empty.

Success Response

This response returns all vizboards defined in Platfora.

HTTP status return code: 200
