POST a Lens Build

Use the POST method to schedule a lens build in Platfora.

URL

/api/v1/lensbuilds

HTTP Method

POST

URL Parameters

None supported.

Data Parameters

None for POST requests.

Sample Call

For POST requests, the body of the HTTP request should be empty.

{
   "data": {
        "lensbuilds": [
            {
              "lens": {
                "id": "578"
              }
            }
        ]
   }
}
