Overview

A dataset is a metadata description of how data is structured. If you are familiar with extract, transformation, and load (ETL) workflows, a dataset makes up the extract and transformation logic in Platfora.

Dataset Attributes

A dataset describes the rows and columns, the base fields and their associated data types, computed dimension fields, measures derived from quantitative data fields, and references (or joins) to other related datasets. The following table summarizes the resource attributes.
