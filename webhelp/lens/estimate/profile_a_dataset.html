Profile a Dataset

You can profile a dataset as long as you have data access and Define Lens from Dataset permissions on the dataset. Profiling a dataset initiates a special lens build to sample the source data and collect its data characteristics.

  1. Go to the Data Catalog and choose the Datasets tab.
  2. Use the List view or the Quick Find to locate the dataset you want to profile.
  3. Choose Profile Dataset from the dataset action menu.
  4. Click Confirm.
  5. To check the status of the profile job, go to the System page and choose the Activities & Lenses tab.

After the profile job is finished, you can review the results that are collected when you build or edit a lens from that dataset. Dataset profile information can only be seen in the lens builder panel, and is only applicable when building aggregate lenses.
