How Data Gets Loaded in Platfora

The way you load data into Platfora is by defining and building a lens. A lens is created by choosing fields from the focal point of a single dataset in the Platfora data catalog.

In Platfora, the dataset is a metadata description of some data residing in Hadoop, along with the processing rules for transforming that data into a structured format that can be used to answer queries. From a dataset, you choose fields that you are interested in exploring. These field choices, along with the range of data you are interested in exploring, can then be saved as a lens.

The act of building a lens in Platfora generates a series of MapReduce jobs in Hadoop to select, process, aggregate, and prepare the data for use by Platfora's query engine. These build jobs result in a lens - a columnar data store that can be used, revised, and discarded as needed. When you work with a lens in a vizboard, the data is loaded into memory as it is needed.
