Build Ad Hoc Visualizations

Using the October 2012 Flight Data lens, you can visually analyze and explore the flights data by building visualizations (viz for short). In Platfora, visualizations are created and managed within a special project document called a vizboard. In this exercise, we will create three common types of viz charts in the Flights Tutorial vizboard: a bar chart, a scatterplot, and a heatmap.

The October 2012 Flight Data lens is an aggregate lens, which is built to support ad hoc data analysis. Ad hoc analysis involves slicing and dicing measure data by different dimensions to discover trends or anomolies in the data.

Viz Builder Basics

A Chart viz shows aggregated measure data grouped and filtered by the chosen dimensions. A chart in Platfora can best be described as a recipe of dimension and measure fields, plus axis placement (X-axis and Y-axis), plus appearance encodings (Color, Size, Shape, Opacity, Labels), plus mark type (Point, Line, Bar, Area, and so on).

  1. Drag fields from the lens data panel to the drop zones in the viz builder panel.
  2. Choose the mark type. Auto (the default) will choose the best mark representation based on the fields in your viz.
  3. Set field Options, such as the display name, number format, or sort order. The field options vary depending on the field role (measure or dimension).
  4. Adjust mark appearance controls, such as color, size, opacity, shape or text labels.
  5. Click the viz title to rename a viz.
  6. From the viz toolbar, you can delete, copy, export, comment, or navigate within a viz.
  7. Drag fields to the Filters drop zone to limit the data shown in the viz.
  • Build a Bar Chart Viz
    A bar chart is used to compare a quantitative value (measure) across one or more categorical dimensions. You can show cumulative dimension values within a single bar by adding an additional dimension to the Color drop-zone. In this exercise, we will build a bar chart using the sample data in the October 2012 Flight Data lens, that compares the number of cancelled flights to non-cancelled flights on the dates surrounding Hurricane Sandy.
  • Build a Scatterplot Viz
    Scatterplots are used to show the correlation of two measures over a common dimension. In this exercise, you will build a scatterplot using the sample data in the October 2012 Flight Data lens, that looks at the correlation of flight departure delays to flight arrival delays on the days surrounding Hurricane Sandy.
  • Build a Heatmap Viz
    Heatmaps are used to compare a measure across multiple dimensions using color to show variations in the data. In this exercise we will build a heatmap to compare the busiest departure and arrival airports in terms of number of flights using the sample data in the October 2012 Flight Data lens.
