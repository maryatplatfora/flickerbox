About the Tutorial Sample Data

Your Platfora installation comes with some sample data that you can use to complete this tutorial. The sample data is FAA flight data from October 2012 (the month of hurricane Sandy). There is a command-line utility in your Platfora server installation to load this data. This utility uploads five csv-formatted source files to the default Uploads data source. The sample data is about 54 MB in size.

The load utility creates four Platfora datasets:

  • Flights (the main fact dataset)
  • Monthly Carrier Activity (another fact dataset)
  • Airports (a dimension dataset)
  • Airline Carriers (another dimension dataset)

The Date and Time dimension datasets come pre-installed with Platfora. These datasets are automatically referenced by any dataset that contains a DATETIME type field.

There is also a source data file available to define an additional Planes dataset.

The load utility also creates and builds one aggregate lens: October 2012 Flight Data. To build this lens, Platfora runs a series of MapReduce jobs in the configured Hadoop source system.

Once the lens build completes, you can use the sample tutorial data to build ad hoc visualizations in Platfora's integrated data analysis workspace, the vizboard.
