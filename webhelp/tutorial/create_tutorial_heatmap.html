Build a Heatmap Viz

Heatmaps are used to compare a measure across multiple dimensions using color to show variations in the data. In this exercise we will build a heatmap to compare the busiest departure and arrival airports in terms of number of flights using the sample data in the October 2012 Flight Data lens.

Add a new page to the Flights Tutorial vizboard.

Create a heatmap using the following chart recipe:

Field Role Field Name Drop Zone / Control Options
    Mark Type Auto or Area
Dimension Arrival Airport .Code X-axis

Sort By=Total Records

Limit=40

Sort Order=Descending
Dimension Departure Airport .Code Y-axis

Sort By=Total Records

Limit=40

Sort Order=Descending
Measure Total Records Color
