Create a User Account

Before you begin working in Platfora, it is best practice to create your own user account and log in as yourself instead of using the default administrator user account. In this exercise, we will add a new user account, grant permissions to this new user, log in as the newly created user, and set your user profile picture.

Step 1: Create a User Account

Every Platfora installation generates a default admin user account so you can log in for the first time (the default password is also admin).

While you are logged in as the default admin user, go to the Platfora System page and create a new user account for yourself. Assign your user account to the System Administrator role so you have access to all of the features in the Platfora application.

  1. Go to the System page.
  2. Select the Users tab.
  3. Click Create User.
  4. In the Create New User dialog, set Name to the desired display name, User Id to the desired login name, Password to the desired password, and System Role to System Administrator.
  5. Click Create.

Step 2: Set Permissions on the Tutorial Datasets

The tutorial datasets are owned by the default admin user. To allow access by other Platfora users, you must grant the appropriate permissions. For this exercise, we will grant ownership access to the default Everyone group. This will allow any Platfora user to work with these tutorial datasets.

  1. Go to the Data Catalog, and select the Datasets tab.
  2. Choose Permissions from the Airline Carriers dataset action menu.
  3. Find the Everyone group, click move to..., and select Own.
  4. Click OK.
  5. Repeat steps 2-4 for the Airports, Flights, and Monthly Carrier Activity datasets.

Step 3: Set Permissions on the Tutorial Lens

The tutorial lens is owned by the default admin user. To allow access by other Platfora users, you must grant the appropriate permissions. For this exercise, we will grant ownership access to the default Everyone group. This will allow any Platfora user to work with this tutorial lens.

  1. Go to the Data Catalog, and select the Lenses tab.
  2. Choose Permissions from the October 2012 Flight Data lens action menu.
  3. Click Add Collaborators.
  4. Select the Everyone group.
  5. Change the Permission from Edit to Own.
  6. Click Add.
  7. Click OK.

Step 4: Log Out and Log In Again

After you have created a user account for yourself, you should log out as the default admin user, and log in again as the new user you just created. That way, you can better track your own activity in Platfora.

  1. In the top right corner of the page header, open the System pull-down menu and select Log Out.
  2. Log in again using the user id and password of your own user account.

Step 5: Add a Profile Picture

Every user account in Platfora can be personalized with a profile picture. Your profile picture is used for your activity updates on the home page and also in your comment posts. If you don't supply your own picture, a default avatar icon is used.

  1. In the top right corner of the page header, select Profile from the current user menu.
  2. In the profile dialog, click the default avatar icon.
  3. Click Select and browse your local file system for an image file in PNG, JPG or GIF format. The default icon size is 160x160 pixels square. Images will be resized and cropped to fit these dimensions.
  4. Click Upload to add the image file to your profile.
  5. Close the profile dialog.
  6. Reload the browser page to refresh the page with the new image.
