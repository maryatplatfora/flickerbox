Security Properties

The security properties allow you to configure some of the various security features Platfora provides, including authentication and SSL access.

  • platfora.dataset.data.access.enabled
    Choose whether or not to control data access at both the data source and dataset. True (eanbled) means it is controlled at both locations, false (disabled) means it is controlled at the data source only.
  • platfora.ldap.connection.timeoutmillis
    Enter the amount of time, in milliseconds, Platfora waits to make a connection with an LDAPS server before it fails the connection.
  • platfora.ldap.response.timeoutmillis
    Enter the amount of time, in milliseconds, Platfora waits for a response from an LDAPS server before closing the connection.
  • platfora.ldap.secure.validateEnabled
    Choose whether or not Platfora validates the LDAPS server secure certificate.
  • platfora.ldap.truststore.location
    Enter the location of the Java trustStore to use when connecting to a secure LDAPS server.
  • platfora.ldap.truststore.password
    Set the password in the Java trustStore for connecting to secure LDAPS servers.
  • platfora.ldap.truststore.type
    Set the type of Java trustStore for connecting to secure LDAPS servers.
  • platfora.ldap.userImportEnabled
    Choose whether or not to users can automatically create a Platfora user by logging into Platfora using their LDAP credentials.
  • platfora.ssl.enabled
    Choose whether or not to allow secure connections to Platfora using HTTPS.
  • platfora.ssl.keystore.location
    Enter the location of the Java keyStore to use for SSL connections.
  • platfora.ssl.keystore.password
    Set the password in the Java keyStore to use for SSL connections.
  • platfora.ssl.keystore.type
    Set the type of Java keyStore to use for SSL connections.
  • platfora.ssl.port
    Enter a port number to use for secure connections using SSL when secure connections is enabled.
  • platfora.ssl.truststore.location
    Enter the location of the Java trustStore to use for SSL connections.
  • platfora.ssl.truststore.password
    Set the password in the Java trustStore to use for SSL connections.
  • platfora.ssl.truststore.type
    Set the type of Java trustStore to use for SSL connections.
  • platfora.usermanagement.cache.expireHours
    Enter how much time, in hours, user or group information exists in the user management cache since its creation before being deleted.
