General Properties

The general properties display some properties that were configured during setup and also allows you to configure basic settings, such as compressing lens files. Note that some general properties descriptions are included in their own sections.

  • platfora.builder.sample.average.row.size.bytes
    Enter the average number of bytes used per row in the Platfora datasets.
  • platfora.builder.sample.cutoff.safety.factor
    Multiplier used as a safety factor to calculate sample mapper row cutoff.
  • platfora.builder.sample.size
    Enter the number of rows Platfora samples from the dataset and retains for dataset profiling.
  • platfora.cors.domain
    Which domains to accept requests from.
  • platfora.cors.enabled
    Indicates if cross-domain header support is enabled.
  • platfora.data.dir
    Local directory Platfora uses to store source from lens builds and various metadata and log files.
  • platfora.dfs.datadir
    Specify a DFS data directory where Platfora build's store its data in Hadoop.
  • platfora.dfs.hive.libdir
    Location of the distributed file system (DFS) directory where Platfora places JAR files for Hive plugins.
  • platfora.dfs.libdir
    Displays the Hadoop DFS directory where Platfora stores its jar files.
  • platfora.field.finder.enabled
    Enables or disables the Field Finder feature.
  • platfora.hive.classpath
    Displays the Hadoop DFS directory where Platfora stores its jar files.
  • platfora.hive.libdir
    Location of a local directory that contains Hive library JARs.
  • platfora.home
    The location on the local disk where Platfora is installed.
  • platfora.jdbc.connection
    JDBC connection URL for metadata catalog. JDBC connection string for the database where Platfora stores its metadata in Postgres.
  • platfora.lens.compression
    Choose whether or not Platfora compresses lens files on disk.
  • platfora.license.expirationwarningdays
    Enter the number of days to display a warning message in the web UI before the Platfora licenses expires.
  • platfora.multinode.nodeId
    ID number of the current Platfora node.
  • platfora.server.management.port
    The port over which the Platfora server and worker nodes communicate with each other.
  • platfora.support.identifier
    Enter an identifier, such as your organization name, that will be used in syscapture file names.
  • platfora.telemetry.aggregate.frequency
    Enter how often to aggregate telemetry data, in seconds.
  • platfora.telemetry.enabled
    Choose whether or not to send system metrics data from the Platfora server to Platfora using HTTPS to improve user experience and performance.
  • platfora.telemetry.file.lifespan
    Enter how long each telemetry file lives on the Platfora master server in seconds.
  • platfora.telemetry.logparser.enabled
    Choose whether or not to include internal Platfora errors with telemetry to help Platfora improve product quality.
  • platfora.telemetry.send.frequency
    Enter how often the Platfora master server sends telemetry data to Platfora, in seconds.
  • platfora.telemetry.url
    Enter the URL to where the Platfora server sends the telemetry data.
  • platfora.webservice.port
    The port used to access the Platfora GUI with a web browser using HTTP.
