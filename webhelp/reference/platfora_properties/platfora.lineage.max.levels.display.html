platfora.lineage.max.levels.display

The maximum height of the tree in the data lineage report represented by the number of node levels as displayed in the Platfora web application.

Full Description

Specifies the maximum number of node levels (tree height) that Platfora includes in data lineage reports displayed in the web application. Each catalog object that is also an object parent represents a new node level. For example, a data lineage report that includes one lens computed field -> one lens base field -> lens -> dataset -> data source has four (4) levels (height of four).

You might want to increase this value if visualizations use multiple derived datasets or computed fields. When this value exceeds the value in platfora.lineage.max.levels.total, the data lineage report limits the number of files to platfora.lineage.max.levels.total. Enter a value of -1 to include all node levels.

Default

5

Value Range/Type

Integer

Configurable in Web Application

Yes

Node Override

No

Restart Requirements

Requires Platfora server restart.
