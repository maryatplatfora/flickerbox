Custom Header and Footer Properties

These properties allow you to configure a custom header and footer in the Platfora application. These are in addition to the application's own header and footer.

  • platfora.footer.dismissable
    Determines if a Platfora user can dismiss the footer's display.
  • platfora.footer.enabled
    Enables or disable display of a custom footer message.
  • platfora.footer.message
    A message to display in the application footer.
  • platfora.header.dismissable
    Determines if a Platfora user can dismiss the header's display.
  • platfora.header.enabled
    Enables or disable display of a custom header message.
  • platfora.header.message
    A message to display in the application header.
