platfora.builder.dictionary.deduplication.cutoff

Enter the maximum input data size above which Platfora dedupes.

Full Description

Specifies a maximum data input size (bytes) for a lens build. When the data involves a higher vale, Platfora builds applies a deduplication step before building dictionaries. The default is equivalent to 100 GB.

Default Value

100000000000

Value Range/Type

Long

Required/Optional

Required

Configurable in Web Application

No, it's not visible

Node Override

No

Restart Requirements

Requires Platfora server restart.
