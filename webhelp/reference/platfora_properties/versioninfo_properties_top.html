Version Information Properties

The version information properties display information about the current installation of Platfora as determined during installation or upgrade.

  • com.platfora.schema.version
    The version of the Platfora database catalog used with postgres.
  • com.platfora.version.build.date
    The data Platfora built this version of the software.
  • com.platfora.version.full
    The version number and the build date.
