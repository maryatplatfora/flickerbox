Lens Builder Properties

The lens builder properties allow you to configure how the lens builder process behaves when building lenses.

  • platfora.build.dictionary.cutoff
    Dictionaries with no more than this number of entries are joined to in-memory in the mapper.
  • platfora.build.dimmap.cutoff
    Dimension key maps with no more than this number of total bytes are joined to in-memory in the mapper.
  • platfora.build.limit.space
    Specify the maximum amount of space Platfora uses on DFS while building a lens, in bytes.
  • platfora.build.limit.time
    Specify the maximum amount of time Platfora uses for building a lens, in seconds.
  • platfora.builder.abort.error.threshold
    The maximum number of lens data quality error rows allowed per lens build. If this limit is exceeded for a lens build job, then the lens build will abort and fail.
  • platfora.builder.chunk.size
    Enter the maximum number of rows that are stored in a single file within a lens.
  • platfora.builder.combine.split.sort.percent
  • platfora.builder.dictionary.deduplication.cutoff
    Enter the maximum input data size above which Platfora dedupes.
  • platfora.builder.dictionary.partitioning.cutoff
    Enter the maximum column cardinality above which Platfora partitions.
  • platfora.builder.dictionary.partitioning.factor
    Enter the number of dictionary partitions Platfora creates for large dictionaries.
  • platfora.builder.encoding.skew.values.max
    Maximum number of skewed values to detect and handle
  • platfora.builder.error.rows.threshold
    The maximum number of lens data quality error rows that will be recorded per map task. If this limit is exceeded, then just that map task will stop writing its data quality log file.
  • platfora.builder.file.skip.modified.age
    Skip the files modified within the last n seconds. This is intended for use with EMR's Simple Server Systems (S3).
  • platfora.builder.global.lens.build.file.limit
    Enter the maximum number of files Platfora creates in HDFS for all lens builds.
  • platfora.builder.incremental.dimensions
    Choose whether or not to perform an incremental lens build instead of a full rebuild when new source files are added to a referenced dataset.
  • platfora.builder.lens.build.concurrency
    Enter the maximum number of lens builds that can be active at the same time.
  • platfora.builder.lens.build.file.limit
    Enter the maximum number of files Platfora creates in HDFS for each lens build.
  • platfora.builder.record.data.quality.errors
    Enables or disables the logging of rows containing lens data quality errors.
  • platfora.builder.record.error.filepaths
    Enables or disables the logging of file path information for each line in a data quality report file.
  • platfora.builder.reduce.max.count.distinct.keys
    Maximum number of unique values in count distinct.
  • platfora.debug.display.lensbuild.warnings
    Choose whether or not to display lens build warnings for successful lens builds in the Platfora web application on the Data Catalog page for lenses and when selecting a lens for a vizboard.
  • platfora.debug.failed.task.log.num
    Enter the number of task log files from failed map and reduce job tasks for each step in a lens build to include in the Platfora server log.
  • platfora.debug.killed.task.log.num
    Enter the number of task log files from killed map and reduce job tasks for each step in a lens build to include in the Platfora server log.
  • platfora.debug.mapper.task.log.num
    Enter the number of map task log files from successful map job tasks for each step in a lens build to include in the Platfora server log.
  • platfora.debug.reducer.task.log.num
    Enter the number of reduce task log files from successful reduce job tasks for each step in a lens build to include in the Platfora server log.
  • platfora.dfs.jobxml.lens.limit
    Number of lenses whose JobXML files we store in the logs directory.
  • platfora.max.corrupt.files.to.skip
    Enter the maximum number of invalid/corrupt compressed source files to skip before failing a lens build.
  • platfora.max.fields.per.job
    Maximum number of fields computed per event series processing job.
  • platfora.max.line.length
    The maximum number of characters allowed for a row in a delimited text-based dataset.
  • platfora.max.pattern.events
    Maximum number of events considered for a single partition in a pattern match expression.
  • platfora.num.errors.reported
    Enter the maximum number of MapReduce task errors, such as skipped source rows, to log in the task log when building a lens.
  • platfora.reduce.tasks
    Enter the maximum number of Hadoop reduce tasks Platfora uses for MapReduce jobs.
