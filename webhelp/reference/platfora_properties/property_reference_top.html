Platfora Configuration Property Reference

Platfora is highly configurable, providing many different properties you can use to fine tune how it performs. You can configure preferences, such as whether or not to compress lenses, and improve performance in your environment based on assistance and feedback from customer support.

  • General Properties
    The general properties display some properties that were configured during setup and also allows you to configure basic settings, such as compressing lens files. Note that some general properties descriptions are included in their own sections.
  • Expressions Properties
    The expressions properties allow you to configure how Platfora handles some built-in functions and user defined functions (UDFs) in its expression language.
  • Lens Builder Properties
    The lens builder properties allow you to configure how the lens builder process behaves when building lenses.
  • Amazon EMR and S3 Properties
    The Amazon EMR and S3 properties allow you to configure how Platfora connects to AWS and initializes an EMR cluster.
  • Advanced Properties
    The advanced properties allow you to configure some advanced internal Platfora processes. Most properties should be changed only when instructed by customer support. Note that some advanced properties descriptions are included in their own sections.
  • Version Information Properties
    The version information properties display information about the current installation of Platfora as determined during installation or upgrade.
  • Security Properties
    The security properties allow you to configure some of the various security features Platfora provides, including authentication and SSL access.
  • Custom Header and Footer Properties
    These properties allow you to configure a custom header and footer in the Platfora application. These are in addition to the application's own header and footer.
