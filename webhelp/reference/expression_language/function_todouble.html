TO_DOUBLE

TO_DOUBLE is a row function that converts STRING, INTEGER, LONG, or DOUBLE values to DOUBLE (decimal) values.

Syntax

TO_DOUBLE(expression)

Return Value

Returns one value per row of type DOUBLE.

Input Parameters

expression
Required. A field or expression of type STRING (must be numeric characters), INTEGER, LONG, or DOUBLE.

Examples

Convert the values of the average_rating field to a double data type:

TO_DOUBLE(average_rating)

Convert the average_rating field to a double data type, but first transform the occurrence of any NA values to NULL values using a CASE expression:

TO_DOUBLE(CASE WHEN average_rating="N/A" then NULL ELSE average_rating END)
