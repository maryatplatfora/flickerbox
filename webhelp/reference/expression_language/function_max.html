MAX

MAX is an aggregate function that returns the biggest value from the given input expression.

Syntax

MAX(numeric_or_datetime_expression)

Return Value

Returns a numeric or datetime value of the same type as the input expression.

Input Parameters

numeric_or_datetime_expression

Required. A field or simple expression of type INTEGER, LONG, DOUBLE, FIXED, or DATETIME. Unlike row functions, aggregate functions can only take simple expressions as input (such as field names or literal values).

Examples

Get the highest value from the sale_amount field:

MAX(sale_amount)

Get the latest date from the Session Timestamp datetime field:

MAX([Session Timestamp])
