About the Default User and Group Accounts

Platfora is installed with a default system administrator account used to log in to Platfora for the first time. Every user account is automatically added to the default Everyone group.

Name User ID Type System Role Description
System Administrator admin user System Administrator (Super) This is the default system administrator account used to log in to Platfora for the first time. The default password for this account is admin. You should change the default password after logging in for the first time.
System system user System Administrator This is a default system administrator account used internally by Platfora during setup. This account does not have login enabled, and it cannot be edited or deleted.
Everyone   group None This is the default group that every user is automatically added to. You cannot edit or delete this group.
