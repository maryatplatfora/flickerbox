Adding a Local User Account

Adding a local user gives a person access to Platfora, and creates an account for them in the Platfora application. A user's system role controls what the user can do in the Platfora application.

  1. Go to the System page.
  2. Go to the Users tab.
  3. Click Create User.
  4. (Optional) Select New Local User. This option is only visible if you also have LDAP authentication enabled.
  5. Enter the following information for the user:
    Field Description
    Name The display name of the user in the Platfora application.
    User Id The user id used to log in to the Platfora application.
    Password The password used to authenticate the user.
    System Role Controls what the user can do in the Platfora application. See Understand Platfora System Roles.
    Add to Group Adds the user to the selected group(s). Every user is automatically added to the default Everyone group.
  6. Click Create.
