Leaving Super Administrator Mode

System Administrators in Super Administrator mode can return to their System Administrator role.

  1. Click the System username at the top of the web application.
  2. From the menu that appears, select Return to System Administrator.

    Platfora removes the red bar at the top of the web application and returns the user avatar to your configured avatar.
