Required HDFS File Permissions

To be granted data access permission on a data source or dataset when HDFS Delegated Administration is enabled, users must have the proper HDFS file permission. This section describes those required permissions.

Users must have read permissions on all files used in an object to perform the following tasks:

  • Create, edit, or view a dataset.

  • Build a lens. When users try to build a lens that uses files they don't have read permissions on, the lens build fails.

  • Profile a lens.

  • Create, edit, or view a vizboard.

Users must have write permissions on a directory to upload a file to that directory.
