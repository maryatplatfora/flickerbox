How Data Access Enforcement Works

Users with the System Administrator role can configure the level at which Platfora allows data access permissions to be set.

Data access can be enforced at the following levels:
  • Data source only. This allows users with the proper permissions to grant other users data access to data sources. This is the default.
  • Data source and dataset. This allows users with the proper permissions to grant other users data access to data sources and datasets. This provides for more fine-grained data access control.

For details on how to enforce data access, see Configure Data Access Enforcement.
