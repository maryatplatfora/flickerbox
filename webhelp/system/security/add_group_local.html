Adding a Group

Groups allow you to organize users by their functional or organizational role in order to centralize the management of permissions.

  1. Go to the System page.
  2. Go to the Users tab.
  3. Click Create Group.
  4. (Optional) Select New Local Group. This option is only visible if you also have LDAP authentication enabled.
  5. Enter the following information for the group:
    Field Description
    Group Name The display name of the group in the Platfora application.
    Description An optional description of the group.
    System Role

    Controls what the users of the group can do in the Platfora application. If a user has multiple system roles or belongs to multiple groups, the most permissive role takes precedence.

    For example, if a user has an individual role of Viewer and they belong to a group with a role of System Administrator, they will have System Administrator system permissions.

    See Understand Platfora System Roles.
    Members Adds the selected user(s) and/or groups(s) to the group.
  6. Click Create.
