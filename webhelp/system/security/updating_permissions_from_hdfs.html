Refresh Data Access Permissions from HDFS

Platfora refreshes the data access permissions based on the current HDFS file permissions at midnight every day and when Platfora restarts. Midnight is determined by the configured time zone for the Platfora master node. System administrators can choose to refresh the data access permissions manually at other times.
Note: Refreshing data access permissions may take a long time, during which system performance may be slow. The time required to refresh the data access permissions depends on the number of datasets and the number of files in each dataset.

  1. Go to the System page.
  2. Go to the Users tab.
  3. Click the System Privileges link.
  4. In the Data Access section, click Refresh Permissions.
  5. Click Continue in the dialog that appears.
  6. After the permissions are updated, click Cancel.
