Lens Contains No Rows

A Lens Contains No Rows warning means that the lens build succeeded, but the resulting lens contains no data. This can happen when lens filters are too exclusive, or if the source data files themselves are empty or corrupt.

A lens can be empty for the following reasons:

  • Lens Filters - The lens has filter conditions that filter out all of the data rows. Edit the lens definition and check the logic of the lens filter expressions.
  • Empty Dataset - The dataset is valid, but there is no source data. This can be caused by empty source data files or by source data files that were moved or renamed in the Hadoop file system. Check the source location that the dataset is pointing to, and make sure that files actually exist in that source location.
  • Invalid or Corrupt Source Data - None of the source data files could be read because they were corrupt or in a file format that Platfora did not recognize.
