Analyze System Activity with the Audit Log

Analyze Activity Using a Log

You can use the platfora-access-audit.log to locate periods of high activity and highly accessed data in your system. This log files contains the following fields:
  • timestamp
  • user
  • audit action (this is always check object permission)
  • PASSed permissions test or MISSed
  • object (dataset, datasource, lens, vizboard)
  • action
The action can be one of the following:
Action Description
CREATE Create an object.
UPDATE Update an object.
DELETE Delete an object. This is only available to an object's owner.
GRANT Grant access to an object. This is only available to an object's owner.
READ_METADATA View the actual object. This is the lowest level of permissions.
UPDATE Update the object.
READ_DATA Read the data behind the object.
  1. Upload the log file.
    1. Log into the Platfora server node and copy the PLATFORA_HOME/logs/platfora-access-audit.log file to a location that is accessible from your Platfora application.
    2. Log into Platfora as a user with Data Administrator permission.
    3. Choose the Data Catalog page.
    4. Press the Add Dataset option.
      You are in the Select Data step.
    5. Press Upload File.
      The Upload a local file dialog displays.
    6. Press Upload to dismiss the dialog and upload the log.
    7. Press Continue to move to the Parse Data step.
  2. Parse the file.
    1. Choose Delimited for the FILE TYPE.
    2. Choose " (quote) for a QUOTE CHARACTER value.
    3. Press Continue to move to the Manage Fields step.
  3. Manage the fields and finish up.
    1. Rename the fields as follows:
      • OriginalDate
      • User
      • Operation
      • Result
      • Object
      • Permission
    2. Hide the Operation field.

      All of the operations are the same -- an operation check.

    3. Choose Add Field and add a computed field with the following value.
      Enter Field name Timestamp
      Expression TO_DATE(OriginalDate,"E MMM dd HH:mm:ss z yyyy")
      At this point, your fields look similar to the following:
    4. Press Continue several times to move to the Finish & Save step.
    5. Give your dataset a name.
    6. Press Finish.
  4. Create a lens with the data.
    1. From the Data Catalog, open your new dataset.
    2. Choose Create Lens.
      The create lens dialog appears.
    3. Enter a Lens name.
    4. Add + (plus sign) next to each visible field. You should leave out Operation as that field is not useful.
    5. Press Build.
      Platfora confirms you want to build the lens. The build can take several minutes depending on your system load. You can use the System page to view the build's progress. When the build completes, you receive a notification.
  5. Create a vizboard and visuzalization.
    1. Select the Vizboards page.
    2. Press Add Vizboard.
    3. Choose your dataset from the Dataset list..
    4. Add the Timestamp field to the Y-AXIS.
      Platfora automatically adds Total Records to the X-AXIS. At this point, you can use the visualization to discover the times when your Platfora server is most busy. In the example, the server was busiest at 2:57:08 in the afternoon.
