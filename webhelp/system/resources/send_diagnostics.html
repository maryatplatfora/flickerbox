Capture and Send System Diagnostics

Diagnostics contain information that the Platfora Support team finds crucial to troubleshooting support issues. To send diagnostics, your Platfora instance must have set platfora.telementry.enabled to true. Administrators can send system diagnostics from the Platfora UI or from a Platfora master's command line.

What Does Diagnostics Capture?

Platfora diagnostics includes the following log files, configuration files, metadata catalog and system environment information:

  • the Platfora master node's log files (up to the latest 50 rotated logs)
  • Platfora's configuration settings
  • the local Hadoop configuration files that Platfora uses
  • a list of files in Platfora's HDFS directory
  • the OS information from the master server
  • memory and CPU system resource data
  • telemetry data if you elected to collect it.

Optionally, Platfora Support may ask you to also make a capture of the Platfora database catalog. Capturing the database catalog requires that you use the command-line tool.

Use the GUI to Capture Diagnostics

Administrators or other users authorized to work with the System page, can send diagnostics from the Platfora application. To send diagnostics:

  1. Log into the Platfora application.
  2. Click the System page.
  3. Choose Send System Diagnostics from the drop down.

    Gathering diagnostics can take several minutes. The dialog displays a Preparing System Diagnostics message while it is gathering information. When the tarball is complete, Platfora automatically sends the tarball to Platfora Support.

  4. Send an email to Platfora support notifying them that you sent a tarball.

Use the Command Line to Capture Diagnostics

The platfora-syscapture command is run from the Platfora master node. The command line utility allows you to send a Platfora database catalog along with the other diagnostic data.

  1. Log into the Platfora master server as the platfora user.
  2. Run the following command to generate a tarball that includes catalog data: 
    $PLATFORA_HOME/bin/platfora-syscapture --with-catalog --child --outfile ~/platfora_syscapture.tar -v
  3. Send the completed tarball to Platfora Support.
