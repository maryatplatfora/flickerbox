Customize the Application Header and Footer

You can add a header and footer to the Platfora application. This feature is useful if you want to display a system-wide message such as an outage announcement.

Administrators can configure a header and/or a footer message for the Platfora application. The properties that control these messages are the following:
  • platfora.footer.dismissable
  • platfora.footer.enabled
  • platfora.footer.message
  • platfora.header.dismissable
  • platfora.header.enabled
  • platfora.header.message
Both messages have a maximum length of 350 characters. The message text can contain HTML markup. The HTML markup counts toward the character limit.

You can configure whether or not users can dismiss either or both of the header or footer. When a user dismisses either message it remains dismissed for all the user's subsequent sessions. Dismissal only applies to the current message. If you configure a new message, the system presents that to the user.

Configuration Procedure

  1. Log in to Platfora as a user with administrator rights.
  2. Select the System tab.
  3. Choose Global Settings from the menu bar.
    The Global Settings dialog displays.
  4. Select Headers and Footers from the sidebar.
  5. Configure the values for the header, the footer, or both.
    For example, you could use a platfora.footer.message similar to the following: 
    Security Access: Access to this system is controlled by S.O.P. 214.5
  6. Press Apply to save your settings.
  7. Restart the server for your changes to take effect.
