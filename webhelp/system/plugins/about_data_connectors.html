About Custom Data Connector Plugins

Platfora natively supports connecting to data sources such as Hive, HDFS, and Amazon S3. For data sources that Platfora does not support out-of-the-box, you can write your own Data Connector plugin using Platfora-provided Java interfaces. Once a custom Data Connector is installed in Platfora, users can use the custom source type to define new data sources in Platfora.

Data Connector plugins must be written in Java using the Platfora-provided Java interfaces. Visit the Platfora Customer Support portal for documentation on these interfaces, a sample data connector plugin implementation, and a tutorial on creating and using a custom data connector plugin.

After writing your plugin, compile it into a single Java archive (JAR) file. This JAR file is then installed and enabled in the Platfora web application.
