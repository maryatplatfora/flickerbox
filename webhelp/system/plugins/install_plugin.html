Install a Plugin

You can extend Platfora functionality by installing a plugin. Once you install a plugin, you cannot remove it.

Platfora expects plugins as compiled JAR (Java Archive) files. To install a plugin, do the following:

  1. Log into the Platfora application.
  2. Click the System tab.
  3. Choose the Plugins tab.
  4. Press the Add Plugin button.
  5. Click Select and navigate to the JAR file.
  6. Click Upload.
    Depending on the plugin, you may need to enter additional data into a plugin dialog.
