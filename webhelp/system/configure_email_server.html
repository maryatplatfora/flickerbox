Configure Platfora to Send Email

To enable Platfora to send email messages, you must configure Platfora to connect as a client to an outgoing email server that uses SMTP (Simple Mail Transfer Protocol).

  1. Go to the System page.
  2. Click Global Settings.
  3. Click Email Configuration.
  4. Enter the following information for the email server Platfora should use for sending email:

    Field

    Description

    Host Name

    The host name of the outgoing email server.

    Port

    The port the email server listens on for email requests. The default port for SMTP over SSL is 465.

    User Name

    The user name of a valid email account on the email server. This email user must be allowed to relay/forward messages. You may want to create a dedicated email account just for Platfora use.

    Password

    The password of the user's email account on the email server.

    Use SSL

    When enabled, Platfora creates a secure connection to the email server using SSL.

    Send as Email

    Enter the email address used to send email notifications. This address is used in the From: and Return-Path email headers.
  5. (Optional) Click Send test e-mail to send an email message to a specified address.
  6. Click Apply Settings.
