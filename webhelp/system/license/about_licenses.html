Understand Licensing

Platfora requires a valid license for users to perform any kind of action, such as creating datasets, editing users, and building lenses. A system administrator uploads a valid license key in the web UI to enable Platfora's functionality.

License keys are stored in a binary file. They contain various information, such as its expiration date and the number of nodes allowed in the Platfora cluster. Users can add as many disabled (inactive) worker nodes as desired without counting toward the number of nodes allowed by the license. However, you can only activate as many nodes as configured in the license. To add or enable more nodes than the current license allows, contact your Platfora sales representative to purchase a license that allows more nodes.

System administrators can add license keys and update existing license keys.

For information on licensing or to contact sales to request a new license, email support at support@platfora.com.
