View License States

Platfora requires a valid license for users to perform any kind of action. Platfora may be in a different state depending on the current status of the license.

The Platfora software can be in one of the following states:

  • Valid license. A valid license key is currently uploaded, and users can perform actions as defined by their assigned roles and permissions. As the date approaches the license expiration date, the web UI displays a notification stating how many more days until it expires.

  • Unlicensed. Platfora has been installed, but no license key has been uploaded to it. In this state, the admin user can log into the web application and accept the end user license agreement (EULA). After accepting the EULA, the admin is redirected to the System > License page to upload a license key. No other pages in the web application are visibile. The web UI displays a notification when you are logged into an unlicensed state.

  • Expired license. A valid license key has previously been uploaded, but the current date is past the license expiration date. Users can log into the web application and view previously created objects, but they cannot create, edit, or build. System administrators can upload a new license key with a later expiration date. The web UI displays a notification when you are logged into an expired license state.
