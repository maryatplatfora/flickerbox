computed field

A computed field generates its values based on a calculation or condition, and returns a value for each input row. Values are computed based on expressions that can contain values from other fields, constants, mathematical operators, comparison operators, or built-in row functions.

Computed fields are useful for deriving meaningful values from base fields (such as calculating someone's age based on their birthday), doing data cleansing and pre-processing (such as grouping similar values together or substituting one value for another), or for computing new data values based on a number of input variables (such as calculating a profit margin value based on revenue and costs). A computed field that does an aggregate calculation is called a measure, which is a special kind of computed field in Platfora.
