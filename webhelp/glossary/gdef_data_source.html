data source

A data source is a connection to a mount point or directory on an external data server, such as a file system or database server. Platfora currently provides data source adapters for Hive﻿, HDFS, Amazon S3, and MapR FS.

Platfora has one default data source named Uploads (for data files that you upload from your local file system). This default data source resides in the distributed file system (DFS) that the Platfora server is configured to use as its primary data source.
