row

A a row represents a single object or record in a dataset. A dataset or lens consists of rows of columns (or fields).

Each row represents a set of related data, and every row has the same structure. For example, in a dataset that represents customers, each row would represent a single customer. Columns might represent things like customer name, email address, gender, age, and so on.
