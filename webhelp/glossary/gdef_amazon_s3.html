Amazon S3

Amazon Simple Storage Service (Amazon S3) is a data storage service provided by Amazon Web Services (AWS).

It is a distributed file system hosted by Amazon where you pay a monthly fee for storage space and data transfer bandwidth. Data transfer is free between S3 and Amazon Elastic Compute Cloud (EC2) clusters, making S3 an attractive choice for users who run Hadoop clusters on AWS or utilize the Amazon EMR service.

Hadoop supports two S3 file system protocols as an alternative to HDFS: S3 Native File System (s3n) and S3 Block File System (s3). Platfora supports the S3 Native File System (s3n) only.
