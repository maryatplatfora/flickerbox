Add Third-party Compression Format Support

If you want to use a third-party compression format that is not one of Platfora's officially supported formats, your system administrator must do some additional configuration to both Platfora and Hadoop.

  1. Log into the Platfora master node as the platfora user.
  2. Make sure you have a $PLATFORA_DATA_DIR/extlib and $PLATFORA_DATA_DIR/extlib/native directory. If you don't create, them.
  3. Obtain the library (.jar file) and native libraries (.so files) for your compression format.
  4. Place JAR compression library in the $PLATFORA_DATA_DIR/extlib directory
  5. Place the native library files (.so files) in the $PLATFORA_DATA_DIR/extlib/native directory.
  6. Repeat step 4 and 5 on each node in the Platfora cluster.
  7. Verify that your Hadoop distribution has the compression format properly installed. If it doesn't, follow your Hadoop vendor's instructions for installing the format.
  8. Restart Platfora.
