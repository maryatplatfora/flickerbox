Change the Sample File for a Dataset

Platfora shows a sampling of 20 records taken from the first file in the source location by default. You can change the sampling file to see a different selection of sample records.

  1. From the Data Catalog page, open the dataset you want to edit.
  2. Click Edit Dataset to open the dataset workspace.
  3. Click Source Data in the dataset header.
  4. Choose another file from the Display sample data using drop-down, and click Refresh Data.
  5. Click Close.
  6. Click Save or Save and Exit.
