Parse Other File Types

For other file types that cannot be parsed using the built-in parsing controls, Platfora provides two generic parsers: Regex and Line. As long as your source data has one record per line, you can use one of these generic parsers to extract columns from semi-structured source data.

  • Parse Raw Lines with a Regular Expression
    The Regex parser allows you to search lines in the source data and extract columns using a regular expression.
  • Parse Raw Lines with Platfora Expressions
    The Line parser simply returns each line in the source file as one column value, essentially not parsing the source data at all. This allows you to bypass the parsing step and instead define a series of computed fields to extract the desired column values out of each line.
