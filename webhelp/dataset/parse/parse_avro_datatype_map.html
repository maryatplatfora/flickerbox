Avro to Platfora Data Type Mapping

Avro has a set of primitive and complex data types it supports. These are mapped to Platfora's internal data types.

Complex data types are imported into Platfora as a single JSON-formatted STRING. You can then use the JSON String Functions in the Platfora expression language to define new computed columns in the dataset that extract a particular name:value pair from the imported JSON structure.
