Parse Raw Lines with Platfora Expressions

The Line parser simply returns each line in the source file as one column value, essentially not parsing the source data at all. This allows you to bypass the parsing step and instead define a series of computed fields to extract the desired column values out of each line.

  1. Open the Parse Data step of the dataset workspace.
  2. Select Line as the File Type in the Parsing Controls panel.
    This creates a single column where each row contains an entire record.
  3. Move to the Manage Fields step.
  4. Define computed fields that extract columns from the raw line.
