Add a New Dataset

In order to add a new dataset to Platfora, the source data files on which the dataset is based must be in the Hadoop file system and accessible to the Platfora application. Platfora takes a sampling of records from a source file and guides you through a series of steps to define the structure and processing rules for the data.

  1. Go to the Data Catalog page.
  2. Click Add Dataset to open the dataset workspace.
  3. Complete the steps in the dataset definition process.
    You do not have to complete the steps in order, and depending on the dataset, you may skip certain steps. To navigate between steps, you can click on a particular step in the left menu or click Continue to go to the next step in the process.
    1. On the Select Data step, select a data source in the Source List. Choose the source data to base the dataset on.
    2. On the Parse Data step, configure the parsing controls to match your source data file format.
    3. On the Manage Fields step, verify your base field information. Add computed fields and measures to describe additional processing rules for the data.
    4. On the Create References step, model the foreign key relationships to other datasets.
    5. On the Define Key step, pick the column(s) that uniquely identify a row or record.
    6. On the Finish and Save step, confirm the dataset name and enter a description.
  4. Click Save and Exit to save the dataset in the Platfora data catalog.
    You can always come back and edit the dataset definition at a later time, however once a dataset is saved, you cannot change its name.
