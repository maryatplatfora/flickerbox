About Dataset Fields

A field is an atomic unit of data that has a name, a value, and a data type. Within a dataset, there are two basic classes of fields or columns: Base and Computed.

Base fields are the raw fields parsed directly from the source data. Computed fields are fields that you add to the dataset to perform some kind of extraction, cleansing, or transformation on the base data fields.

A measure is a special type of computed field that aggregates quantitative data. Measures are the reporting metrics, and every dataset must have at least one. The result of a measure computation is determined at lens build time depending on the dimension groups selected for a lens.
