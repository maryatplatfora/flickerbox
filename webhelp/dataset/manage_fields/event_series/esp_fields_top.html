Event Series Processing Computed Fields

You can add an event series processing computed field to a Platfora dataset. Event series processing is also referred to as pattern matching or event correlation.

Computed fields that use the PARTITION function are event series processing fields. Use event series processing (ESP) to partition rows, order the rows sequentially (typically by a timestamp), and search for matching patterns among the rows.

Instead of computing values from the input of a single row, ESP fields compute values from inputs of multiple rows in the dataset. The PARTITION function is not an aggregate function. The function returns one value (or column) per row. You can use an event series processing field in other expressions or (after the lens builds) in a viz.
