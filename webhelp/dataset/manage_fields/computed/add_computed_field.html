Add a Computed Field

You add a computed field by writing an expression that transforms existing fields. Computed fields help you refine your data processing.

Computed fields contain expressions that can take other fields as input. These fields can be base fields or they can be other computed fields.

  1. Go to the Manage Fields step of the dataset workspace.
  2. Click Add Field in the Field Info panel.
    This opens the Add Field dialog containing the expression builder controls.
  3. Enter a name for your field and a description.
    The description is optional but very useful for others that might use your dataset later.
  4. Choose function from the Functions list.
    Use the drop-down to restrict the type of functions you see.
  5. Double-click a function from the list to add it to the Expression area.
    The Expression panel updates with the function's template. Also, the Fields list refreshes with those fields you can use with the function. For example,TO_DATE works on STRING data types.
  6. Double-click a field to add it into the Expression area.
  7. Continue adding functions and fields into your expression is complete.
  8. Make sure your expression is correct.
    The system checks your syntax as you build the expression. The yellow text box below the Expression area displays error messages. Platfora only allows you to save expressions that evaluate successfully. If it cannot resolve an expression, the Save button is not available.
  9. Click Save to add the new computed field to the dataset.
    Your new computed field appears as a new column in the dataset.
  10. Check the computed column values to make sure the expression logic is working as expected.
    Computed fields that use aggregate functions are special and cannot display sample values. See Add Computed Measures for more information about these types of computed fields.

Expressions are an advanced topic. For information on working with expression syntax, see Work with Platfora's Expression Language.
