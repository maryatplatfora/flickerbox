About Binned Fields

A binned field is a special kind of computed field that groups ranges of values together to create new categories. Binned fields are a way to reduce the number of values in a high-cardinality column, or to group data in a way that makes it easier to analyze. For example, you might want to bin an age field into categories such as under 18, 19 to 29, 30-39, and 40 and over.
  • Bin Numeric Values
    You can bin numeric values by adding a binned quick field.
  • Bin Text Values
    If you wanted to bin text or STRING values, you can define a computed field that groups values together using a CASE expression.
