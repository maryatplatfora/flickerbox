Drill Paths

Adding a drill path to a dataset allows vizboard users to drill down to more granular levels of detail in a viz. A drill path is defined in a dataset by specifying a hierarchy of dimension fields.

For example, a Product drill path might have categories for Division, Type, and Model. Drill path levels depend on the granularity of the dimension fields available in the dataset.
