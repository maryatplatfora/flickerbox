Edit a Dataset

Defining a dataset is often an iterative process. You can edit a dataset definition at any time to modify its descriptions, source location, parsing logic, and processing rules. However, you cannot change the name of a dataset once it has been saved the first time.

  1. Go to the Data Catalog page.
  2. Click the dataset to open it.
  3. Click Edit Dataset.
  4. Go to the step in the dataset definition that you want to edit.
    To navigate between steps, click on a particular step in the left menu of the dataset workspace.
    1. The Select Data step is disabled when editing a dataset. See Change the Source File Location for a Dataset or Change the Sample File for a Dataset.
    2. The Parse Data step is limited when editing a dataset. You can change the parsing controls to match your existing source data file format. However, it is currently not possible to pick up additional columns or schema changes made in the source data after the dataset has been created.
    3. The Manage Fields step is most likely where most dataset edits will occur. You can add or edit computed fields and measures to modify the processing rules for the data.
    4. On the Create References step, model the foreign key relationships to other datasets.
    5. On the Define Key step, pick the column(s) that uniquely identify a row or record.
    6. On the Finish and Save step, edit the dataset description.
  5. Click Save and Exit to save your changes.
