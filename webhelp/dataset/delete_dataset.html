Delete a Dataset

Deleting a dataset from Platfora removes the dataset definition from the Platfora data catalog.

Deleting a dataset does not remove files or directories in the source file system. It does not remove lenses built from the dataset. Any lenses that have been built from the dataset will remain in Platfora, however future lens builds that use a deleted dataset will fail.
  1. Go to the Data Catalog page.
  2. Click the dataset to open it.
  3. Click Delete Dataset.
  4. In the confirmation dialog, click Delete.
