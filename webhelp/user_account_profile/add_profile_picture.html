Add a User Profile Picture

Every user account in Platfora can be personalized with a profile picture. Your profile picture is used for your activity updates on the home page and also in your comment posts. If you don't supply your own picture, a default avatar icon is used.

  1. In the top right corner of the page header, select Profile from the current user menu.
  2. In the profile dialog, click the default avatar icon.
  3. Click Select and browse your local file system for an image file in PNG, JPG or GIF format. The default icon size is 160x160 pixels square. Images will be resized and cropped to fit these dimensions.
  4. Click Upload to add the image file to your profile.
  5. Close the profile dialog.
  6. Reload the browser page to refresh the page with the new image.
