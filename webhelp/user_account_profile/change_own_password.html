Change Your User Password

If your Platfora user credentials are managed within Platfora, you can change your own user password from your user profile page. If your system administrator has configured Platfora to use an external LDAP server for authentication, then you would change your password in that external system, not in Platfora.

  1. In the top right corner of the page header, select Profile from the current user menu.
  2. In the profile dialog, click Change Password.
  3. Enter your new password. Make sure to type carefully as there is no password confirmation!
  4. Click Update Password.

If you made a mistake entering your password and get locked out of Platfora, a Platfora system administrator can reset your password for you from the Platfora System page.
