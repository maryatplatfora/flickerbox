IAM User for Platfora

AWS Identity and Access Management (IAM) allows you to create users and groups and control access to AWS services and resources. Platfora recommends creating an IAM user account specifically for use by Platfora. This user must have (at a minimum) the permissions in the AmazonElasticMapReduceFullAccess security policy.

You can create a new platfora user in the IAM Management Console of your AWS account. After you have created the user, download the AWS credentials for this user. You will need the Access Key Id and Secret Access Key when you initialize Platfora for use with Amazon EMR.

Under Permissions for this user, attach the user policy for Amazon Elastic MapReduce Full Access. This policy provides full access to Amazon Elastic MapReduce and underlying services that it requires such as EC2 and S3.
