EC2 Security Group Settings

EC2 security groups allow you to specify firewalling rules for your Amazon elastic cloud computing (EC2) server instances.

EC2 security group rules are independent of, and in addition to, the software firewalling provided by the instance's operating system. Security groups must be defined before you create an EC2 instance.

The security group configured for the Platfora server instance must permit connections from your user network to the Platfora web application server port (8001 by default). You also may want to open the EMR Hadoop JobTracker and TaskTracker web ports so that you can monitor and troubleshoot MapReduce jobs executed by Platfora.
