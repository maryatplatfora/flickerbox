The Platfora Application Site Map

Platfora is a web application, meaning users can access it from any computer with network access. There is no client software to install; all you need is a supported web browser. The Platfora web application has three main functional areas: Data Catalog (data ingest), System (system administration), and Vizboards (data analysis),.

On the Data Catalog pages, data administrators define the datasets that reveal Hadoop data to users. Data administrators and analysts also use the Data Catalog to create lenses.

On the System pages, system administrations manage the users, groups, and data access permissions for Platfora. They can also connect data sources, configure global installation settings, and manage lens builds.

On the Vizboard pages, users that have analyst privileges build visualizations and group them in project dashboards called vizboards.
