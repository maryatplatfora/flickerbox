Export Viz Data Lineage

Exporting data lineage to a JSON file for all fields in a visualization allows data analysts to see where the data in the fields ultimately came from and how they were manipulated in the process. The data lineage report includes filters applied to fields used in the viz.

  1. From the viz toolbar click the export menu and select Download Lineage as JSON.
  2. The JSON file (platforaData.json by default) is saved in the default downloads directory configured for your browser.
