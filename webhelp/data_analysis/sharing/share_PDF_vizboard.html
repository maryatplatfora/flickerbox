Export Vizboard as a PDF

You can export a vizboard as a PDF file. You might want to do this to share insights discovered in Platfora with a larger group of people. You can share rendered PDFs with other Platfora users or users without access to Platfora. Users do not need access to a vizboard to view a rendered PDF.

PDF rendering happens in the background. Depending on the size and number of visualizations, this might take some time. You can continue working in the vizboard while the PDF is rendered. You can view the progress of PDF rendering jobs on the System > Activities & Lenses page.

When the rendering is complete, the file is downloaded to the default downloads directory for the browser and a notification is displayed in the Platfora UI. Platfora stores generated PDF files on the server. Platfora retains PDF files for one day by default (your administrator might change this value). You can manually download a PDF while it still exists on the server. To do this, click the link to the file from your user profile notification list. Once it's been deleted, you must render the PDF again.

Note: By default, Platfora allows vizboard users to export to PDF. However, your administrator may disable this functionality.
  1. Click the vizboard Share menu.
  2. Choose Prepare PDF for Download.
    Note: You can also choose Prepare PDF for Download from the vizboard contextual menu on the Vizboards page.
  3. Click Save Vizboard and Prepare PDF or Prepare PDF, depending on whether the vizboard was already saved.
    Platfora saves the vizboard, if applicable, and begins to render the vizboard as a PDF file. When the PDF is ready, it's downloaded to your local machine according to how the web browser is configured.
    Note: Large vizboards may take some time. Platfora informs you when it starts rendering a PDF and when the file is ready.
  4. (Optional) You can download the PDF manually from your notification center.
