Change the Value Range of a Measure Axis

By default, the range of values shown on a measure axis always includes 0 (zero) in the range. For data that has a range skewed to high values, you may not want the measure axis to start at zero. For example, if your data values start at 1,000,000 and go to 10,000,000 then you may want the measure axis to reflect the actual range of values rather than starting at 0.

When you have a measure field in the axis drop zones (X-axis or Y-axis), the default is to always include 0 in the range of values on the axis. To have the axis reflect the actual range of values, you can choose to not always include 0 in the range.

Note that this will only affect measure fields whose range does not include 0 already and where the range is skewed. If the range naturally includes 0 (or values close to 0), then the measure axis must include 0.

  1. Select Options in the measure field drop-down menu.
  2. Deselect Always include zero under the Range options.
  3. Click Confirm.
For some data, it is easier to see the variations in the data when the range starts at a higher value rather than at zero.
