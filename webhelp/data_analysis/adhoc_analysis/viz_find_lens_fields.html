Find Fields in a Lens

When you pick a lens to work with, the measure and dimension fields included in the lens are shown in the lens data panel. The data panel does not show all of the dataset fields, only those that were requested when the lens was built.

Measure fields () are always listed at the top, followed by dimensions ( ) and referenced dimensions () in alphabetical order. Computed fields that were defined in the vizboard are highlighted blue.

If you hover over a field name, you can see a tooltip with the field details and the field option menu.

In a long list, you can use the search to filter fields by name. Clear the search criteria to remove the search filter.
