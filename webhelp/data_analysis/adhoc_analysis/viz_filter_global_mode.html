Designate Page Filters

Designating a page filter allows you to filter multiple visualizations at once (provided they all contain the same field). For example, if you designate a page filter on a Date field in one visualization, then any other visualization on the same vizboard page that uses Date will have that filter applied as well.

To make a filter apply to the page:

  1. Select the visualization that contains the filter.
  2. Find the field in the filters panel.
  3. Open the filter Options menu.
  4. Select Apply to entire page.

If any other visualizations on the same vizboard page also contain the field, the filter will be applied to those visualizations as well.
