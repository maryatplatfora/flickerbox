Create Solid Shapes

Marks on a Point chart appear as hollow shapes by default. This section describes how to change the marks to solid shapes.

  1. Enable separate controls for the color and opacity attributes by selecting Color and Opacity from the Marks pull-out menu.
  2. Place a dimension field in the Fill Color drop zone.
  3. In the Stroke Opacity pull-out menu, change the opacity to 0%.
