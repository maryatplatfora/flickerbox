About Cross-Tab View

If you want to see the data comprising the points on a visualization, you can toggle the viz to Cross-Tab view. This shows the data in tabular, spreadsheet format.

In Cross-Tab view, measure values are always shown in columns, regardless of where the measure fields are placed in the Builder panel drop zones. Dimension fields placed in the Rows drop zone will show as rows in Cross-Tab view. However, dimensions placed in other drop zones such as Details, Color or Shape will sub-divide the measure columns.

  • Enable Cross-Tab Totals
    Cross-Tab view can be enabled to show totals for the data in columns and rows. For example, if each row shows the sum of orders, then turning on totals for rows creates a column showing the sum of all orders in each row.
