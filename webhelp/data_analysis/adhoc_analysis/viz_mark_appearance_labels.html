Adjust Mark Labels

Viz marks can be displayed with or without a label representing one of different values associated with each mark.

The Label menu controls the text that applies to all marks in the viz.

You can choose from the following label options:

  • Auto- When a field is in the Label drop zone, this option displays the value of the drop zone field for each mark. When no field is in the Label drop zone, Auto displays no label text.

  • X-Axis Value - This option always displays the mark's value along the horizontal axis. You might want to use this option to quickly and easily read the mark's exact value on the horizontal axis without zooming in.

  • Y-Axis Value - This option always displays the mark's value along the vertical axis. You might want to use this option to quickly and easily read the mark's exact value on the vertical axis without zooming in.

  • None - This option turns off the text labels.
