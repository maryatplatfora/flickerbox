Change Mark Appearance Options

Visualization mark appearances can be changed by color, size, opacity, shape, and label text. You can also increase the number of appearance controls to more finely tune the appearance of a viz. This allows you to increase the data density of a viz through mark appearance.

By default, all marks are represented by a single color, size, and opacity level. However, you can split the controls for each of these attributes into separate outline (stroke) and fill controls. For example, you can create a Point chart with solid shapes with a separate color for the outline, or no color for the outline.

Use the pull-out menu for the Marks drop zone the to enable separate controls for color, size, and opacity.

When separate controls are enabled for an appearance attribute, two drop zones are available for that attribute instead of one. For example, the Color drop zone is replaced with the Stroke Color and Fill Color drop zones.
