Choose the Analysis Type

Every visualization performs one type of analysis, such as ad hoc (chart or cross-tab) or funnel. The analysis type determines the type of lenses available for the viz, such as aggregate or event series lenses. Choosing an analysis type shows all datasets that have lenses of the appropriate type built from the focal point of that dataset.

With a viz selected, choose the analysis type in the data panel.
Datasets that have lenses available for the selected analysis type are listed in alphabetical order. If you do not see the dataset you want, that means that no lenses of the appropriate type have been built for that dataset.
Note: When you choose either ad hoc analysis option, you can change to other type in the visualization. For example, if you choose Cross-Tab here, you can change the viz to Chart at any time.
