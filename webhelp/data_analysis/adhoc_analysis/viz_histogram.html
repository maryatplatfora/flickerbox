Creating a Histogram

Marks on a Bar chart are separated by empty space by default. This section describes how to change a bar chart to a histogram (no space between bars).

  1. In a bar chart viz, enable separate controls for the size attributes by selecting Size from the Marks pull-out menu.
  2. In the Fill Size pull-out menu, maximize the size by changing it to 1.00x.
