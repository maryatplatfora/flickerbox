Create Funnel

To create a funnel analysis visualization, you first choose an event series lens and then define the stages in the funnel.

  • Choose the Analysis Type
    Every visualization performs one type of analysis, such as ad hoc (chart or cross-tab) or funnel. The analysis type determines the type of lenses available for the viz, such as aggregate or event series lenses. Choosing an analysis type shows all datasets that have lenses of the appropriate type built from the focal point of that dataset.
  • Define Funnel Stage
    Define funnel stages in the stage builder of a viz of the funnel analysis viz type.
  • Analyze Funnel Viz
    Funnel visualizations show the total number of users who reach each stage. You can also analyze funnels by comparing funnels across dimension fields.
