Define Funnel Stage

Define funnel stages in the stage builder of a viz of the funnel analysis viz type.

  1. Enter a name for the stage.
  2. Choose an event defined in the lens.
  3. (Optional) Define a condition for the stage by first choosing a field and then the condition that the users must meet to reach this stage.
    Platfora lists all datetime and dimension fields in the event and dimension dataset. The event or reference name appears after the field name. This is useful when different datasets have fields with the same name.
  4. (Optional) Click the icon to add another stage condition.
  5. (Optional) Click Create New Stage to add another stage to the funnel. Or, click the icon to duplicate a stage and then edit the copy.
    You can change the order of a stage in the funnel using the and icons to move a stage up or down.
