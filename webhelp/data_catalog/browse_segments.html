Find Available Segments

The data catalog does not have a centralized tab or view that shows all of the segments that have been created in Platfora. You can find segments by looking in a particular dataset to see if any segments have been created from that dataset.

  1. Go to the Data Catalog.
  2. Find the dataset that you would want to segment and open it.
  3. Choose the Segments tab.

    If you do not see a Segments tab on the dataset details page, this means the dataset is not eligible to define segments. A dataset must be the target of an incoming reference in order to define segments from it.

    In this release, segments cannot be defined from the data catalog. Segments can only be defined in the vizboard from a lens where the key of this dataset is included in the lens as a referenced field.

  4. Click a segment to see its details or action menu.
  5. The segment details shows:
    Segment Name The name given to the segment when it was defined in the vizboard
    Segment Dataset The segment dataset is the same as the currently selected dataset
    Segment Count The key members of this dataset that met the segment conditions defined in the vizboard.
    Created By The time the segment was defined and the user who created it.
    Modified By The time the segment was last refreshed and the user who last updated it. Segments can be refreshed from the System page.
  6. From the segment action menu you can Delete the segment or edit its Permissions.

    To view or edit the segment conditions, you must go to a vizboard where the segment is in use. Choose Show Vizboards to find vizboards that are using the segment.
