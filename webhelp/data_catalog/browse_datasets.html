Find Available Datasets

Datasets represent a collection of source data in Hadoop that has been modeled for use in Platfora. You can browse or search the data catalog to find datasets that are available to you and that meet your data requirements. Once you find a dataset of interest, you can request that data by building a lens (or check if a lens already exists that has the data you need).

Search within Datasets

Using the Quick Find search, you can find datasets by name. Quick find also searches the field names within the datasets.

  1. Go to the Datasets tab in the Data Catalog.
  2. Search by dataset name or by a field name within the dataset using the Quick Find search.

Dataset List View

List view allows you to sort the available datasets by different criteria to find the dataset you want.

  1. Go to the Datasets tab in the Data Catalog.
  2. Select List view.
  3. Click a column header to sort by that column.
  4. Once you find the dataset you want, use the dataset action menu to access it.
  5. While in list view, you can select multiple datasets to delete at once.
